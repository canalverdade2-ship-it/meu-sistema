# Scripts Automáticos de Sincronização do Git / GitHub

Esta pasta contém scripts de 1 clique para atualizar, baixar e enviar alterações para o GitHub.

## 🚀 Como usar:

### 1. `ENVIAR_SUBIR_GITHUB.bat` (Push)
- Adiciona todas as alterações feitas no projeto (`git add .`).
- Pede uma mensagem de descrição da alteração.
- Envia os arquivos atualizados para o GitHub (`git push origin main`).

### 2. `BAIXAR_ATUALIZAR_GITHUB.bat` (Pull)
- Baixa todas as novidades e atualizações que estão no GitHub para o seu computador (`git pull origin main`).

### 3. `SINCRONIZAR_TUDO_GITHUB.bat` (Sync Completo)
- Baixa o que há de novo no GitHub, grava suas alterações e envia tudo atualizado em um único clique.
